CSS

CSS describes how HTML elements are to be displayed on screen, paper, or in other media. Some of the main properties are :

    Borders

    This property allows us to set borders of different style to any component of the webpae, be it a division, section, table, or anything else.

    Margins

    This property allows us to set size of the space around the elements as per our wish. It has 4 options : margin - top, margin - bottom, margin - left, margin-right.

    Paddings

    This property allows us to generate space round content. It is somewhat similar to margin, and also has 4 options : padding - top, padding - bottom, padding - right, padding - left.

    Fonts

    This property allows us to set the font sie, color, style and font variant.

    Links

    There are 4 types of links in HTML. First is the normal, unvisited link. Second is, the visited link. Third is called hover link, and the last one is active link. We can set different styles for all four, including different color, size etc so that its easy for a person to make out which link belongs to which category.

    Align

    This property lets us align text, images, sections etc both relative to other content or otherwise.

    Dropdowns

    CSS dropdowns allow the user to write text such that when hovered over, it shows some description of the text.
